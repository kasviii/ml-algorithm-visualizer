# ML Algorithm Visualizer

An interactive, browser-based tool to visualize core Machine Learning algorithms step by step.

## Live Demo
[https://kasviii.github.io/ml-algorithm-visualizer](https://kasviii.github.io/ml-algorithm-visualizer)

## Algorithms Covered

**Supervised Learning**
- K-Nearest Neighbors (KNN) — adjustable K, live neighbor radius
- Linear Regression — gradient descent with live loss tracking
- Decision Tree — step-by-step splits using Gini impurity
- Support Vector Machine (SVM) — margin maximization visualization

**Unsupervised Learning**
- K-Means Clustering — centroid initialization, assignment, and convergence
- Principal Component Analysis (PCA) — variance direction and dimensionality reduction

**Ensemble Methods**
- Bagging / Random Forest — bootstrap sampling and majority vote boundary

## Features
- Step-by-step execution with explanations at each stage
- Auto-run mode for full animation
- Live parameter controls (adjust K, learning rate, etc.)
- Real-time metrics panel (loss, iterations, votes, variance explained)
- Works entirely in the browser — no installation, no backend

## Tech Stack
- Pure HTML, CSS, JavaScript
- Canvas API for rendering
- Deployed via GitHub Pages
